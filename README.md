Economic Dashboard (Power BI)

Project Overview
My project presents an interactive Power BI dashboard analyzing key economic indicators in Poland like: average wages, real wages (adjusted for inflation), CPI inflation, unemployment rate.

I used Tools & Technologies like: Python (pandas), Power BI, CSV data processing.

Data Processing
Raw data was cleaned and transformed using Python scripts.  
Multiple datasets were merged into a single analytical table used in Power BI.

Dashboard Features
- time-series analysis
- interactive slicers (year selection)
- comparison of economic indicators
